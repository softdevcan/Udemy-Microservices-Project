# Udemy-Microservices-Project
The following tools were used in the project:
- ASP .NET Core 7.0
- Microservices Architecture
- Databases:
  - MSSQL
  - PostgreSQL
  - MongoDB
  - Redis
- IdentityServer
- AutoMapper
- Dapper
- Domain Driven Design
- CQRS
- RabbitMQ with MassTransit
- Docker and Portainer
